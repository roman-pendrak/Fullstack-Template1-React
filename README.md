Modern React full stack template using latest technologies :zap:

# Install Node 18+, clone, build, and run!
Node Version Manager For Windows: https://github.com/coreybutler/nvm-windows/releases
```bash
# build
$ npm install

# run dev mode
$ npm run dev

# production build
$ npm run build

# run production build
$ npm run preview
```
